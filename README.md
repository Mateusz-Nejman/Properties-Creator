# Properties-Creator
Program that can create javascript object or simple json file. This app can be used as language file editor. It can create one-layer json file.
<br>Example json file:
```javascript
{
    "name" : "Mateusz",
    "surname" : "Nejman",
    "age" : "21",
}
```
<br>Example javascript object:
```javascript
{
    name : "Mateusz",
    surname : "Nejman",
    age : "21",
}

```
Small Difference :)
